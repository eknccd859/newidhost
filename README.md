## Things Worth Knowing Before You Buy

**Multiple data center locations.** For the VPS and dedicated plans, you get actual geographic choice — US, UK, Germany, Netherlands, Romania, Australia, Singapore. That's meaningful if your audience is geographically specific and you care about latency.

**LiteSpeed on the NVMe cPanel plans.** Apache is fine, but LiteSpeed is noticeably faster for most CMS workloads. Getting it on the shared plans is a decent value-add.

**7-day money-back guarantee on cPanel hosting.** Not the industry-standard 30 days, so read the terms before committing if you're unsure. The individual/blogger plans reportedly carry a 35-day guarantee based on forum listings.

**Fully managed WordPress.** The managed WP plans handle backups and security — that's not always included at this price tier with some competitors, where "managed" can mean surprisingly little.
